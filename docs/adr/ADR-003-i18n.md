# ADR-003 — Internacionalización (i18n)

**Estado:** Aceptado  
**Fecha:** 2026-07-17

---

## Contexto

La web debe soportar español y catalán desde el inicio, con arquitectura preparada para inglés, alemán, francés e italiano.

---

## Decisión

Se usa el sistema nativo de Laravel (`lang/`) con prefijos de locale en las rutas:

```
/es/...
/ca/...
/en/...   (preparado)
/de/...   (preparado)
/fr/...   (preparado)
/it/...   (preparado)
```

---

## Consecuencias

- Ningún texto visible se escribe directamente en las vistas Blade.
- Los archivos de traducción residen en `lang/{locale}/`.
- El middleware `SetLocale` detecta el locale desde la URL.
- Los metadatos SEO se traducen por locale.
- Las fechas se formatean con `Carbon` según el locale activo.
